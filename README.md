# Inventory Management

## Description
A **frontend-only Inventory Management** application with **user authentication**, **dashboard statistics**, and **product listing** using static JSON data.  
Built to demonstrate **client-side logic**, **responsive UI**, and **clean structure** without any backend.

---

## Features
- **Login with Email & Password**
  - Client-side validation
- **Static Authentication**
  - Uses LocalStorage
  - Auto-redirect if user is already logged in
- **Dashboard**
  - Logged-in user information
  - Logout functionality
  - Product statistics:
    - Total Products
    - In-Stock Products
    - Out-of-Stock Products
- **Product Management**
  - Responsive product cards
  - Product details view (no routing library)

---

## Tech Stack
- **HTML** – Semantic & accessible
- **Tailwind CSS** – Responsive Grid & Flexbox
- **JavaScript** – DOM manipulation, validation

---


## Known Limitations
- Frontend-only authentication
- Static product data
- No backend or routing library
